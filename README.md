# liveBoard
微信小程序留言板


## 目的

为自己的公众号实现留言功能，增加和用户的互动性！现在的架构只支持单用户部署。就是自己新建一个小程序，不适合多用户共用。

## 主要功能

- [x] 1、管理员可以为文章增加删除留言
- [x] 2、可以对用户留言进行多轮回复
- [x] 3、管理员可以删除、置顶留言
- [x] 4、用户可以对留言进行点赞
- [x] 5、用户可以订阅管理员回复
- [x] 6、管理员可以接手用户留言微信通知。
- [x] 7、可以一键跳转文章
